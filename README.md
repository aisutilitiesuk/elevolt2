# Elevolt Investment Presentation

Single-page presentation web app (8 slides) for hosting on Vercel.

## Deploy on Vercel

1. **From this folder** (recommended):
   ```bash
   cd presentation
   npx vercel
   ```
   Or connect the repo in [Vercel](https://vercel.com) and set **Root Directory** to `presentation`.

2. **As standalone**: Copy the contents of `presentation/` (e.g. `index.html`, `styles.css`, `vercel.json`) into a new repo and deploy that repo.

No build step required; Vercel will serve the static files.

## Local preview

Open `index.html` in a browser, or use a local server:
```bash
cd presentation
npx serve .
```
